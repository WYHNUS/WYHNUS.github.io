20分钟用NodeJS和AngularJS搭建网站

Posted on 2017-03-13

最近几年网页开发越来越容易上手，近半年来写了好几个网页的demo，觉得用 NodeJS 做后端，AngularJS 写前端的开发速度很快，并且不需要在多语言间来回切换，很适合用来快速做网页的MVP(最简可行产品)。

以下教程适用于 NodeJS 和任意前端框架组合 (如 React.js, Vue.js等)

大纲：

  1. 安装node.js和npm (3 minutes)
  2. 了解文件夹结构 (3 minutes)
  3. 安装依赖模块 (2 minutes)
  4. 编写 home.jade (2 minutes)
  5. 编写 node.js 的入口文件 (5 minutes)
  6. 编写 AngularJS 的入口文件 (5 minutes)

2. 文件夹结构 (3 minutes)

先上图:

解释：

  1. demo : 是我们要做的网站根目录的名字
  2. demo/node_modules : 网站依赖模块所在的文件夹 (无需自行创建，之后会自动生成)
  3. demo/public : 前端代码的根目录
  4. demo/public/app : AngularJS的根目录 (和Angular有关的前端代码都在这里开发)
  5. demo/public/app/components : Angular的每个components在此目录下新建一个文件夹放view和controller
  6. demo/public/app/components/home/homePage.html : Angular模块的范例，主页的html (MVC 的 View)
  7. demo/public/app/components/home/homePageController.js : Angular模块的范例，主页的控制器 (MVC 的 Controller)
  8. demo/public/app/app.js : AngularJS的入口文件
  9. demo/public/stylesheets : 前端的css文件
  10. demo/views : 服务器端渲染的html文件根目录
  11. demo/views/home.jade : 入口文件 (jade是一种模版引擎，不知道的话可以简单理解成html)
  12. demo/.gitignore : 如果项目在Github上备份的话，这里记录了不想同步的文件
  13. demo/app.js : NodeJS的入口文件
  14. demo/package.json : 设置文件
  15. demo/README.md : 项目的描述
